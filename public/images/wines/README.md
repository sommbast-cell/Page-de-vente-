# Photos des bouteilles

Place ici les photos de tes bouteilles au format JPG.

**Nommage** : chaque photo doit porter le `slug` exact du vin (défini dans `data/wines.json`), avec l'extension `.jpg`.

Exemples :
- `chateau-montalbret-renaissance-2023.jpg`
- `chateau-montalbret-chloe-2022.jpg`
- `primitivo-doppio-passo-2025.jpg`
- `magnum-m-minuty-rose-2025.jpg`
- etc.

**Format recommandé** :
- JPG, 800 × 1200 px
- Fond blanc ou neutre
- 200 à 400 ko par image

Tant qu'une image n'existe pas, le site affiche automatiquement un placeholder SVG élégant selon la couleur du vin.
